The constructor of ConvexPolyhedron shape.

The shape MUST be convex for the code to work properly. No polygons may be coplanar (contained in the same 3D plane), instead these should be merged into one polygon.

Parameters
idId of the ConvexPolyhedron
pointsAn array of Vec3's
facesArray of integer arrays, describing which vertices that is included in each face.
uniqueAxesPtrAn pointer to array of Vec3's, describing the unique axes of this shape. default is nullptr
