Cannon CPP —— C++ 3D physics engine

Cannon Cpp is a C++ port of Cannon.js.

Cannon.js is a lightweight 3D physics engine for the web.

Introduction

Cannon Cpp is a C++ port of Cannon.js with some modifications:

  • Add a reference counting mechanism.
  • Add a memory pool management.
  • Events are dispatched by the dispatcher of the world.
  • Remove PreStep and PostStep Event for each body.
  • The way to create a heightfield with an image is not supported.

Cannon Cpp was originally used as the C++ 3D physics engine for Hago Game Runtime.

Example

// Create a world
auto world = new cannon::World();
world->set_broadphase(new cannon::NaiveBroadphase());
world->set_allowSleep(true);
world->get_defaultMaterial()->set_friction(0.5);
world->get_defaultMaterial()->set_restitution(0);
// Create a ground material
cannon::Material::MaterialOptions matOptions(world->genMaterialId());
auto groundMaterial = new cannon::Material(matOptions);
groundMaterial->set_friction(0.3);
// Create a ground plane
auto groundShape = new cannon::Plane(world->genShapeId());
// Create a ground body
cannon::Body::BodyOptions groundBodyOptions(world, world->genBodyId());
groundBodyOptions.mass = 0;
groundBodyOptions.material = groundMaterial;
auto groundBody = new cannon::Body(groundBodyOptions);
groundBody->addShape(groundShape);
world->addBody(groundBody);
// Create a box shape
float size = 1;
cannon::Vec3 halfExtents = cannon::Vec3(size, size, size);
auto boxShape = new cannon::Box(world->genShapeId(), world, halfExtents);
// Create a box body
cannon::Body::BodyOptions boxBodyOptions(world, world->genBodyId());
boxBodyOptions.mass = 1;
auto boxBody = new cannon::Body(boxBodyOptions);
boxBody->addShape(boxShape);
boxBody->get_position().set(0,0,5);
world->addBody(boxBody);

Build for WebAssembly

Install Emscripten:

$ git clone https://github.com/emscripten-core/emsdk.git
$ cd emsdk
$ git pull
$ ./emsdk install latest
$ ./emsdk activate latest
$ source ./emsdk_env.sh
$ emcc -v
  • Set your_emsdk_path/upstream/emscripten to $EMSCRIPTEN in PATH env

Compile Wasm:

$ cd webassembly
$ $EMSCRIPTEN/emmake make
  • The Cannon_v1.0.0_min.wasm.wasm will be output to dir webassembly/build

WebAssembly usage for Cocos Creator3D In <a href="https://open.ihago.net/">Hago Game Runtime</a>:

  • Copy Cannon_v1.0.0_min.wasm.js & Cannon_v1.0.0_min.wasm.wasm to your project build output dir
  • Replace main.js with main-fix-hago.js
  • Add <script src="Cannon_v1.0.0_min.wasm.js" /> to index.html
  • Replace cocos3d-js.js with the code snippet:
    cannon_min = window.WASM_CANNON || cannon_min;
    setWrap & getWrap
    shapesLength & getShapeOffset & getShapeOrientation
